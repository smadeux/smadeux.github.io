Christmas Light Controller

This is a system I built to contol up to four strings of christmas lights

The brains of this controller is a Raspberry Pi that sends signals to its GPIO pins based on which strings of lights shoud be on and which should be off. The GPIO pins are connected by jumper wires to a 4-channel relay module and each channel of the relay is connected to an extension cord. Each string of christmas lights just needs to be plugged into the corresponding extention cord and you're good to go.


This project started when I took an analog circuits class. I decided build the basis version of this where four strings of lights could be turned on and off by a python script. After the class was over, I decided to add the ability for the lights to blink in coordination with music.

The digram above shows how the GPIO pins were connected to the relay module to control the lights as well as one pin for power(5V) and another for ground.
